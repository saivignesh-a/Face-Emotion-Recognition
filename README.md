Objectives :
To build a deep learning model that recognizes emotions from facial images
To preprocess and organize image data for training and testing
To train a CNN model for accurate emotion classification
To evaluate model performance using accuracy and loss metrics

step - 1 git clone https://github.com/your-username/Face-Emotion-Recognition.git

step - 2 Install required libraries:
pip install tensorflow keras numpy pandas opencv-python matplotlib

step - 3 Open the Jupyter Notebook:
jupyter notebook Face_proj.ipynb

step - 4 Run all cells to train and test the model
